Dropdown Component Demo

This demonstrates a reusable dropdown menu built with two Frame components, featuring cross-frame animations and interactions.

Features

  • Two Frame Components: Trigger and menu are separate Frame components
  • Cross-Frame Animation: Smooth transitions between open/closed states
  • Click Outside to Close: Automatically closes when clicking elsewhere
  • Keyboard Navigation: Escape key closes the dropdown
  • Positioning: Supports multiple positions (bottom-left, bottom-right, etc.)
  • Customizable: Full Frame API available for styling
